yNodes Plugin

Welcome to the yNodes Plugin documentation!

yNodes is a comprehensive town-and-nation Minecraft plugin (Spigot/Paper) that lets players create towns, form nations, claim territory, wage wars, engage in diplomacy, and manage economies. It is a fork of the Phonon Nodes plugin, updated and expanded with rich features like territory control, flag war system, alliances/peace treaties, plots, religion/ideology mechanics, chat channels, and more.

Features Overview

  • Towns & Nations

    Players can found towns, invite residents, and unite towns into nations for mutual benefit. Towns claim land chunk-by-chunk as territories and can ally or fight with others. Nations consolidate member towns under one banner and define diplomacy and chat channels.

  • Territorial Claims

    Land claiming is chunk-based starting from a home territory. Claims expand with /town claim and can be released with /town unclaim. A map view and optional minimap help visualize nearby territories.

  • Economy & Resource Nodes

    Territories generate resources which accumulate in a Town Income inventory accessible via /town income. Admins can trigger temporary ore bonuses with /orebonus.

  • Flag War System

    During war, players place a flag in enemy territory to start a conquest timer. If successful, the chunk is captured. War mode is opt-in and tracks a War Score for ongoing conflicts.

  • Diplomacy – Alliances, Truces, Treaties

    Towns and nations can ally using /ally, negotiate truces, and finalize peace treaties via an interactive GUI started with /peace.

  • Religion & Ideology Systems

    Players may found or join religions and ideologies using in-game menus. Towns and nations can set official beliefs that persist across sessions.

  • Land Plots & Chest Protection

    Residents can create personal plots within town territory, manage trusted players, and transfer ownership while keeping chests protected.

  • Custom Chat Channels

    Switch between global, local, town, nation, and ally chat channels for focused communication. Commands like /townchat and /nationchat toggle each channel.

  • Name Tag Customization

    Town prefixes and suffixes appear before or after player names and can integrate with PlaceholderAPI and ProtocolLib for enhanced name plates.

  • Dynmap Integration

    When Dynmap is present, yNodes outputs JSON data to render town territories on a live map with color-coded overlays.

  • PlaceholderAPI Support

    Register placeholders such as %nodes_town% or %nodes_prefix% for use in other plugins if PlaceholderAPI is installed.

  • Data Storage & Backup

    All data is saved in JSON format with optional auto-backups. Commands allow manual save or load operations for safety.
